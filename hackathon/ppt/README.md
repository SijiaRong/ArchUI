## 概述

PPT 模块负责黑客松演示文稿的全部规范，包括叙事结构、每页内容要点、视觉风格与排版约束。

## 典型幻灯片结构

1. **封面** — 项目名称、一句话描述、团队信息
2. **问题** — 痛点陈述，数据支撑
3. **解决方案** — 核心功能演示
4. **差异化** — 与现有方案的关键区别
5. **市场规模** — TAM / SAM / SOM
6. **Demo** — 产品截图或现场演示入口
7. **团队** — 成员与分工
8. **下一步** — 路线图与需求

## 设计约束

- 每页文字不超过 30 字
- 主视觉配合 `image-design` 模块生成的素材
- 字体、色值与 `brochure` 保持一致
