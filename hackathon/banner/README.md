## 概述

易拉宝模块负责黑客松现场竖版展架（Roll-up Banner）的设计规范，适用于 80cm × 200cm 标准尺寸。

## 内容分区（从上到下）

| 区域 | 高度占比 | 内容 |
|---|---|---|
| **品牌顶栏** | 15% | Logo、活动名称 |
| **主视觉** | 35% | Gemini 生成的核心图片 |
| **项目名称** | 10% | 大号字体，一句话 tagline |
| **三大亮点** | 25% | 图标 + 短句，三列排布 |
| **二维码 / 联系** | 15% | Demo 链接、团队联系方式 |

## 制作规范

- 尺寸：800px × 2000px（1:2.5），300 DPI 印刷输出
- 出血：四边各留 3mm
- 主色调与 `brochure` 保持一致
- 主视觉图片来自 `image-design` 模块
