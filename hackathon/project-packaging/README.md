## 概述

项目包装是所有展示材料的内容基础。本模块通过三个子模块将项目的核心论点结构化，确保 PPT、视频、宣传册等输出物的内容口径一致。

## 子模块

| 模块 | 职责 |
|---|---|
| `problem-statement` | 解决什么问题 — 痛点定义、目标用户、问题量化 |
| `competitive-analysis` | 与现有方案对比 — 功能矩阵、差异化优势 |
| `market` | 市场分析 — 规模估算、增长趋势、切入策略 |

## 使用规则

展示材料中涉及"问题是什么""为什么我们更好""市场有多大"的内容，必须以本模块为唯一来源，不得在各材料中各自表述。
