## 概述

视频模块负责黑客松演示视频的全部规范，目标是在 2–3 分钟内完成项目故事的完整讲述。

## 视频结构

| 段落 | 时长 | 内容 |
|---|---|---|
| **开场钩子** | 0–15s | 一个真实痛点场景，引发共鸣 |
| **问题陈述** | 15–30s | 现有方案的不足，数据量化 |
| **解决方案展示** | 30s–90s | 产品核心功能 screencast |
| **差异化亮点** | 90s–2min | 3 个关键优势，快速切换 |
| **Call to Action** | 2min–结尾 | Demo 链接、联系方式、下一步 |

## 技术规范

- 分辨率：1920×1080（16:9）
- 时长：2–3 分钟
- 配音：普通话，语速适中，配字幕
- 背景音乐：轻音乐，不盖过人声
- 封面帧：使用 `image-design` 模块生成的主视觉
