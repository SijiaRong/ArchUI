## 概述

宣传册是黑客松现场发放的品牌材料，兼顾快速浏览（封面+亮点）和深度阅读（内页详情）两种场景。

## 折页结构（三折页，6 面）

| 面 | 内容 |
|---|---|
| **封面** | Logo、项目名、一句话 tagline、主视觉图 |
| **封底** | 二维码、官网/联系方式、团队名 |
| **内页左** | 问题陈述 + 痛点数据 |
| **内页中** | 解决方案核心功能（3 点，图文并茂） |
| **内页右** | 差异化优势 + 竞品对比简表 |
| **内页背面** | 市场规模 + 团队介绍 |

## 设计规范

- 尺寸：A4 展开（297mm × 210mm），三折后每面 99mm × 210mm
- 色彩：遵循 `vi` 模块定义的品牌色系
- 字体：遵循 `vi` 模块定义的字体规范
- 图片：主视觉来自 `image-design` 模块
- 输出格式：印刷版 PDF（300 DPI）+ 数字版 PDF（72 DPI）
