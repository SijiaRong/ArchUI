## 概述

宣传硬卡是黑客松现场发放的品牌材料，正面快速传达项目定位，反面展示核心亮点与联系方式。A5 硬卡单手可持、便于携带，适合评委和观众快速了解项目。

## 尺寸与材质

- 尺寸：A5（148mm × 210mm）
- 出血：四边各 3mm（成品裁切后 148×210mm）
- 材质：300g 铜版纸覆哑膜（推荐）或覆亮膜
- 圆角：四角 3mm 圆角（可选）

### 正面文案

**痛点：**
> 代码和文档是割裂的 — 文档写完就过时，AI 啃长文档效率低，知识无法复用。

**解决方案：**
> ArchUI 把文档拆成模块树，每个模块独立可理解、可复用。人类用画布编排模块关系，AI agent 按需加载上下文，直接生成代码。

## 设计规范

- 色彩：遵循 `vi` 模块 — 主色 `#F07040`、白底、文字 `#1a1a2e` / `#3d3f52` / `#7a7d90`
- 字体：遵循 `vi` 模块 — Noto Sans SC + JetBrains Mono
- 图形语言：圆角卡片、橙色渐变分割线、极淡网格背景，与易拉宝风格一致
- 主视觉：节点画布 SVG，与 `banner` 模块同源
- 二维码：与 `banner` 模块同一个演示链接 QR 码
- 输出格式：印刷版 PDF（300 DPI）+ 数字版 PDF（72 DPI）
