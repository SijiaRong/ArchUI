## 概述

VI（Visual Identity）模块是整个 Hackathon 展示体系的设计根基。所有子模块（PPT、易拉宝、宣传册、图片生成）的视觉决策均以本模块为单一来源，保证跨材料的品牌一致性。

### 图片构图原则
- 主体居中或遵循黄金比例
- 留白不少于画面面积的 30%
- 避免过于繁杂的背景

## Logo 使用规范

- 最小尺寸：40px 高度（屏幕）/ 10mm 高度（印刷）
- 安全距离：Logo 高度的 50%
- 禁止变形、旋转、更改颜色
- 深色背景使用白色版，浅色背景使用彩色版
